# KYB 常见问题：企业认证

*【预计阅读时间：3分钟】*

本文介绍了 Bitget 了解你的企业（KYB）验证流程，包括所需文件类型、高管角色选择、不同地址情况、受限国家，以及常见的验证注意事项。

**1.什么是企业认证？**

企业认证是“了解你的企业（KYB）”措施的一部分。企业认证可以帮助金融机构验证机构用户及其业务相关的合规风险。

**2.为什么要进行企业认证？**

在我们的生态系统中，企业认证（KYB）在建立安全环境以保护用户免受欺诈、洗钱和非法活动侵害方面发挥着至关重要的作用。这种做法不仅值得推荐，也是监管机构当作风险缓解和合规措施而大力提倡和执行的标准要求，在打击洗钱和确保有效风险评估方面尤其如此。通过实施企业认证，我们可以提高识别和降低潜在风险的能力，为营造更安全、更值得信赖的商业环境做出贡献。

**3.企业认证（KYB）需要哪些文件？**

以下要求条目仅供参考。请在企业认证时选择最相符的“组织类型”，以查看适用于贵企业的具体要求内容。

a.公司注册证书

b.公司查册报告/注册代理人出具的《董事在职证明》或政府机构/公共网上登记处提供的《商业登记摘要》（或同等效力文件），文件出具时间应在12个月以内，且需包含最新的董事及股东信息。

c.公司章程或任何具有同等效力的章程文件

d.显示所有中间实益拥有人和最终实益拥有人所有权分布的所有权结构图，由一名董事签字并注明日期（模板如下）

![KYB 常见问题：企业认证](https://img.bgstatic.com/multiLang/web/6c7b33366553d9ca54cf984745ac1481.png)

e.所有最终实益拥有人、终极控制人、董事和被授权人的护照或身份证件

f.过去3个月内出具的所有最终实益拥有人/终极控制人的住址证明

g.3个月内出具的经营地址证明

**4.如何确定在“关联高管”板块选择哪些角色？**

以下是“关联高管”板块中每个角色的简要定义，以及一些示例：

a.董事是监督实体运作并有能力代表实体签署文件的个人。在大多数情况下，董事身份可通过最新的公司查册报告/企业注册摘要或同等资料进行核实。

b.最终实益拥有人（UBO）是指持有实体25%或以上股份或投票权的个人。他们可能不直接管理企业，但最终会从资产、收入或利润中获益。

c.获授权人是指被授权以履行业务职责并有权代表实体签署文件的个人。在 Bitget 企业认证中，获授权人是指被授权代表入驻实体并操作 Bitget 企业账户的个人。董事通常是企业默认的代表，但他们也可以通过董事会决议或授权书将这一权力下放给获授权人。

d.终极控制人是指拥有实体最高控制权的人。大多数情况下，最终实益拥有人与终极控制人之间的主要区别在于，最终实益拥有人注重经济上的所有权和利益，而终极控制人则注重法律和经济上的控制权。即使没有 UBO，如实体属于基金会或其他没有明确所有权结构的实体，也必须确定控制人。这类情况下，实体可能没有直接的财务受益人，但仍必须明确谁最终有权控制和指导实体的运营。

例1：

• 如果个人身兼多职，则应在“关联高管”板块为其选择所有适用的职务。

例2：

•如果小红是 UBO，小明是董事兼获授权人，则在“关联高管”板块应为两者选择相应的角色，具体来说，小红应选择最终受益人（UBO）角色，小明则应选择“董事”和“授权代表”角色。这样可以确保角色指定准确，清晰反映每个人与实体之间的关系。

请注意，所提供的示例仅供参考。角色的实际选择可能因企业结构的复杂性或实体类型而异。

**5.如果我在实体中担任多个角色，我应该选择哪一个？**

在“关联高管”板块，每个人可选择多个角色。详细说明请参阅《[企业认证分步指南](https://www.bitget.com/zh-CN/support/articles/12560603802558)》中的第5步。

**6.我能否指定一名员工或代理人为获授权人？**

如果要指定一名员工或代理人作为获授权人，代表实体操作账户，则需要一份由董事正确签署并注明日期的董事会决议或授权书。

**7.如果实体更名，需要哪些文件？**

需要提供名称变更证明，以核实名称变更。文件应显示原名称、新名称和更改生效日期。

**8.哪些文件可以作为地址证明？**

以下是 Bitget 接受的地址证明列表。除非另有说明，否则文件**日期应在3个月内**，并应寄给有关个人或实体。

a.公用设施账单（电费、水费、网费、电话费等）

b.银行对账单

c.政府出具的居住证明（如人口系统的电子摘要）

d.保险公司发出的信函

e.**最新**报税表/市政税账单/税务发票

您的住址不应是邮政信箱地址，除非这是您所在辖区唯一可用的地址形式。

**9.我可以使用注册地址作为“办公地址”吗？如果我在家远程经营业务怎么办？**

您需要提供经营地址（或办公地址）来证明您实际存在。以下是一些示例情景供您参考。

场景1：

• 实体租用或拥有一间办公室，其地址用于公司注册，同时也是企业运营的场所。在这种情况下，经营地址与注册地址相同。

场景2：

• 注册地址与秘书服务机构或注册代理人的地址相同。但是，由于该地址仅用于您的代理人进行注册，您需要在申请时提供您的实际办公地址。此外，您还必须提交最近3个月内的相关地址证明。

情景3：

• 注册地址与提供智能办公服务（如 WeWork）的公司秘书服务机构地址相同。您可以使用此地址作为您的经营地址，但您必须提供地址证明，表明您所在企业与公司秘书服务机构之间的关系（例如，公司秘书服务机构向您出具的有效办公室租赁协议/公用设施发票）。

情景4：

• 如果企业居家经营，您需要填写您的居住地址作为办公地址，并提供近3个月内的相关地址证明。

**10.为什么我无法在国家列表中找到我所在的国家？**

Bitget 不向以下国家/地区的用户提供服务：

加拿大、克里米亚、顿涅茨克、卢甘斯克、古巴、香港、印度、伊朗、朝鲜、新加坡、苏丹、美国（包括以下美国领土：波多黎各、关岛、美属维尔京群岛、美属萨摩亚群岛和北马里亚纳群岛以及下列美国本土外小岛屿：贝克岛、豪兰岛、贾维斯岛、约翰斯顿环礁、金曼礁、中途岛、纳瓦萨岛、帕尔米拉环礁和威克岛）、伊拉克、利比亚、也门、阿富汗、中非共和国、刚果民主共和国、几内亚比绍、海地、黎巴嫩、索马里和南苏丹。

**11.在哪里可以看到企业认证失败的原因？**

申请通过审核后，您注册的邮箱地址将收到一封系统邮件，如果您的申请被拒绝，邮件中将显示拒绝原因。您还可以访问企业认证页面查看拒绝原因。
