# Bitget 账户提交企业认证需要哪些文件？

*【预计阅读时间：3分钟】*

本文详细介绍了完成 Bitget 账户企业认证（KYB）流程所需的文件。

**什么是企业认证？**

企业认证是了解企业用户流程的一部分，可确保 Bitget 准确验证机构账户。这一过程对于评估合规风险和维护交易平台的安全性至关重要。

企业认证通过保护用户免受欺诈、洗钱和其他非法活动的侵害，建立一个安全的环境。作为一项标准监管要求，它有助于降低风险、确保合规性并支持有效的风险评估。通过实施企业认证流程，Bitget 增强了其生态系统内的透明度、信任度和安全性。

**企业认证所需文件**

企业认证所需的文件可能因组织类型而异。确保在认证过程中选择正确的“组织类型”，以便查看具体要求。

1. **公司注册证书**：证明公司成立的法律文件。
2. **公司查册报告或企业注册摘要：**需由政府相关部门或注册代理机构出具，且在近12个月内有效。
3. **董事在职证明：**需由注册代理人出具，且在近12个月内有效。
4. **组织大纲和章程**：概述公司结构和宗旨的公司文件。
5. **最新股东名册**：所有股东的最新名单。
6. **最新董事名册**：所有董事的最新名单。
7. **所有权结构图**：由董事签署并注明日期，且显示中间受益所有人和最终受益所有人的结构图。该模板将在企业认证过程中提供。
8. **身份证明文件**：最终受益所有人、董事和授权人的护照或政府颁发的身份证件。
9. **住址证明**：最终受益所有人和控制人的近期证明文件（最近3个月内出具）。
10. **经营地址证明**：最近3个月内的公司经营地址证明文件。

*注意：确切的文件要求取决于认证时所选择的实体类型。*

**有关企业认证的其他信息**

- **实体名称变更**：如果企业名称发生变更，请提供一份名称变更证明，注明原名称和新名称以及变更日期。
- **地址证明**：可接受的文件包括公用事业账单、银行对账单、政府官方出具的证明以及税务文件。日期必须在最近三个月内。
- **公司地址**：包括公司的实际经营地址。避免使用邮政信箱或注册代理地址，除非您所在的司法管辖区允许此种做法。

**常见问题1. 我可以使用注册代理人的地址作为经营地址吗？**

不能，您应提供企业的实际经营地址。

**2. 如果我在家远程经营业务怎么办？**

如果企业居家经营，您需要填写您的居住地址作为办公地址，并提供近3个月内的相关地址证明。

**3. 地址证明文件的时效性要求是什么？**

地址证明文件的日期必须在最近三个月内。

**4. 是否接受文件的扫描件？**

是的，一般可以接受高清扫描件。请确保所有信息清晰易读。

**5. 如果我的企业更名了怎么办？**

请提供一份名称变更证明，注明原名称和新名称，以及变更生效日期。

**6. 如果我还有其他问题怎么办？**

如需帮助，请通过**kyb@bitget.com**联系我们的客服团队。
