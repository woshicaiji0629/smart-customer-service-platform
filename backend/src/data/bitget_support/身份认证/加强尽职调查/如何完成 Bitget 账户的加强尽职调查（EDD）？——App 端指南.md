# 如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南

*【预计阅读时间：3分钟】*

本文**详解**在**Bitget App 端**完成加强尽职调查（EDD）流程的步骤。

## 什么是加强尽职调查（EDD）？

加强尽职调查（EDD）是针对高风险用户、交易或账户活动所需完成的额外验证流程，具体包括提供更详细的信息和证明文件，以验证**用户资金来源**、**财富来源**和**交易行为的合法性**。

完成 EDD 有助于确保您的账户安全，交易顺畅，并支持 Bitget 继续为所有用户提供安全的交易环境。

## 如何在 Bitget App 端完成 EDD？

### 第1步：登录您的 Bitget 账户。

1. 登录您的 Bitget 账户，然后点击位于左上角的**个人资料图标**。

2. 选择“**个人资料”**下的“**身份认证**”。

3. 点击“**去认证**”开始尽职调查流程。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/a246524b2bb815bb1a33d13c71f7c8f0.jpeg)

### 第2步：上传地址证明

1. 点击“**下一步**”开始验证您的地址证明。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/67d5180c40d456599d680bc1966bcee6.jpeg)

2. 请从下拉菜单中选择您的**居住国家/地区**。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/9d0cfd4ab081e14737a818ef1fbecb91.jpeg)

3. 请上传您的**地址证明**。有效地址证明文件日期必须在3个月内。例如：

• 水电煤气费账单

• 银行对账单

• 信用卡账单

• 纳税申报

• 网络、有线电视或电话账单

• 市政税单

• 政府签发的居住证明等。

4. 请在“备注”栏中补充其他信息（如有）。

5. 点击“**下一步**”继续。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/6f624b5e5f9f0c6d5e639ba0f2ce3774.jpeg)

### 第3步：上传资金来源信息

1. 请从下拉菜单中选择您的主要资金来源。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/e596503af1643d6b78bba7f9c5c4470b.jpeg)

2. 请上传相关证明文件以证明您的资金来源。

3. 请在“备注”栏中补充其他信息（如有）。

4. 点击“**下一步**”继续。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/3c560ef28e9c342538bcc180746e8a09.jpeg)

### 第4步：上传财富来源

1. 请从下拉菜单中选择您的主要财富来源。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/d07ecee5331ab50fe5c2bf0ec3a72886.jpeg)

2. 请上传相关证明文件以证明您的财富来源。

3. 请在“备注”栏中补充其他信息（如有）。

4. 点击“**提交**”完成您的 EDD 申请。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/641409bf1cebc4106bf0ff1cbd64605f.jpeg)

### 第5步：提交审核

1. 完成所有步骤后，您的申请状态将变为**“审核中”**。

2. 如认证获批或需补充信息，您将收到邮箱通知，处理通常需要**1–2个工作日。**

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/1279551c11b4dee041b66afb0de870ab.jpeg)

### 第6步：查看申请状态

您可以随时登录 Bitget App 并访问身份认证页面来查看您的申请状态。如需补充文件或信息，我们将把申请退回给您，并在退回申请中用红色标明拒绝理由。我们还会向您注册账户的 Bitget 邮箱地址发送通知，其中包含详细的拒绝原因列表，供您参考。

![如何完成 Bitget 账户的加强尽职调查（EDD）？——App 端指南](https://img.bgstatic.com/multiLang/web/2ff85012ea1af2d5fa858592377ed918.jpeg)

• 您可以在 Bitget App 重新提交 EDD 申请时查看拒绝原因。

## 常见问题解答

**1.什么是加强尽职调查（EDD）？**

加强尽职调查（EDD）是针对高风险客户、交易或账户活动所需的额外尽职调查级别。其流程包括收集更详细的信息，以确保用户资金和交易行为的合法性。

**2.我为什么需要完成 EDD？**

EDD 是遵守反洗钱/反恐怖主义融资法规的要求，并旨在确保您所使用的资金的合法性。该流程有助于保护您和 Bitget 的合规标准。

**3.上传文件有哪些格式和大小要求？**

支持的格式：JPG、PNG、PDF。如果文件超过 20MB，请将其分割成较小的文件并分别上传。

**4.EDD 审核需要多长时间？**

如果所有材料清晰完整，EDD 审核通常会在提交后1–2个工作日内完成。

**5.如果我没有按时提交 EDD 会有何后果？**

如果未在规定时间内完成 EDD，您的某些账户功能（如交易）可能会暂时受限，直至成功完成 EDD。

**6.提交的文件必须为英文吗？**

我们也接受其他语言的文件，只要关键信息清晰易读即可。除非 Bitget 合规部门在特殊情况下特别要求，否则无需提供认证翻译件。

**7.我可以遮挡文件中的敏感信息吗？**

您可以遮挡不相关的敏感数据（如无关交易），但您的姓名、居住地址、账户号、文件签发人以及相关交易记录等关键信息必须完全可见。过度遮挡可能会导致文件遭拒。

**8.我可以提交文件的截图吗？**

屏幕截图需保证所有必需信息均清晰可见，才可接受。
