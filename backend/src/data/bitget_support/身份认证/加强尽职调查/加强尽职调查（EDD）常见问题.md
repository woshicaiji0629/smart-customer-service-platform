# 加强尽职调查（EDD）常见问题

*【预计阅读时间：5分钟】*

本文解答关于加强尽职调查（EDD）的常见问题，并说明 EDD 审核流程中您会遇到的情况。

**1.什么是加强尽职调查（EDD）？**

加强尽职调查（EDD）是针对高风险客户、交易或账户活动所需的额外尽职调查级别。其流程包括收集更详细的信息，以确保用户资金和交易行为的合法性。

**2.我为什么需要完成 EDD？**

EDD 是遵守反洗钱/反恐怖主义融资法规的要求，并旨在确保您所使用的资金的合法性。该流程有助于保护您和 Bitget 的合规标准。

**3.哪些文件可以作为地址证明？**

除非另有规定，可接受的地址证明必须以您的名义开具，且开具日期在三个月内。可接受证明文件包括：

a.公共事业账单（电费、水费、网络/电话费等）

b.银行对账单

c.政府出具的居住证明（例如人口系统电子摘录）

d.保险公司出具的函件

e.最新的报税单/市政税账单/税务发票

**4.EDD 可接受/需要提供哪些文件？**

可接受的文件因您的主要资金来源或财富来源而异。您需提交与主要资金来源或财富来源相对应的证明材料。请参考以下可接受文件清单。

| 资金来源（SOF） | 佐证文件示例 |
| --- | --- |
| 储蓄存款 | 银行对账单 |
| 工资 | 工资单、银行对账单、税务文件 |
| 奖金和分红 | 工资单、银行对账单、税务文件 |
| 经营收入 | 财务报表或利润表/损益表 |
| 投资 | 投资/证券账户对账单、银行对账单、股票凭证 |
| 赠予 | 书面证明，说明资金来自朋友、家人或其他主体的赠与 |
| 借贷 | 贷款流水 |
| 继承 | 遗嘱认证书、或继承文件 |
| 房地产 | 按揭/抵押文件、房产交易记录、产生租金收入的租赁协议 |
| 资产出售或出租 | 资产出售协议、产生租金收入的租赁协议 |
| 彩票中奖 | 彩票兑奖证明 |
| 其他来源 | 与离婚、诉讼、赔偿等相关的文件 |

| 财富来源（SOW） | 佐证文件示例 |
| --- | --- |
| 就业 | 简历、工资单、税务文件 |
| 经营收入 | 财务报表或利润表/损益表 |
| 投资 | 投资/证券账户对账单、银行对账单、股票凭证 |
| 赠予 | 书面证明，说明资金来自朋友、家人或其他主体的赠与 |
| 继承 | 遗嘱认证书、或继承文件 |
| 资产出售或出租 | 资产出售协议、产生租金收入的租赁协议 |
| 彩票中奖 | 彩票兑奖证明 |
| 其他来源 | 与离婚、诉讼、赔偿等相关的文件 |

**5.上传文件有哪些格式和大小要求？**

支持的格式：JPG、PNG、PDF。如果文件超过 20 MB，请将其分割成较小的文件并分别上传。

**6.EDD 审核需要多长时间？**

如果所有材料清晰完整，EDD 审核通常会在提交后**1–2个工作日**内完成。

**7.如果我没有按时提交 EDD 会有何后果？**

如果未在规定时间内完成 EDD，您的某些账户功能（如交易）可能会暂时受限，直至成功完成 EDD。

**8.准备文件时如有疑问，可以联系客服吗？**

可以。如需协助，您可以[提交工单](https://www.bitget.com/zh-CN/ticket/ticket-form?ticketId=2061678148075433984)联系 Bitget 客服，并提供与问题相关的详细信息。我们的客服团队将审核您的请求，并尽快协助您解决问题。

**9.提交的文件必须为英文吗？**

我们也接受其他语言的文件，只要关键信息清晰易读即可。除非 Bitget 合规部门在特殊情况下特别要求，否则无需提供认证翻译件。

**10.我可以遮挡文件中的敏感信息吗？**

您可以遮挡不相关的敏感数据（如无关交易），但您的姓名、居住地址、账户号、文件签发人以及相关交易记录等关键信息必须完全可见。过度遮挡可能导致文件拒收。

**11.如果我的文件遭拒收怎么办？**

您将收到通知，说明需澄清或补充的内容。请在规定的截止日期之前，使用可接受的文件重新提交 EDD。

常见拒绝原因包括：

• 图片模糊或不完整

• 缺页

• 文件日期超过三个月

• 文件信息与 Bitget 账户信息不一致

**12.我可以提交文件截图吗？**

屏幕截图需保证所有必需信息均清晰可见，才可接受。

**13.如何加快 EDD 流程？**

为避免延误，请：

• 上传高质量且完整的文件

• 确保上传文件上的姓名与身份认证记录一致

• 上传能证明所选主要资金来源/财富来源的佐证文件

• 及时响应 Bitget 合规团队的补充材料请求

**14.我的个人信息安全吗？**

我们会采取适当的加密措施保护所收集的用户数据。所有个人数据均依据严格的信息安全与数据保护标准以及适用法律进行处理。

**15.我可以提交多份佐证文件吗？**

可以，提供额外佐证文件有助于我们更好地了解您的资金来源与财富来源，并可能加快核验流程。
