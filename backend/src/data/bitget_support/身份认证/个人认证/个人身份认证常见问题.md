# 个人身份认证常见问题

*【预计阅读时间：5分钟】*

本文解答用户关于Bitget**个人身份认证的常见问题**，并提供有用信息。

**1.为什么要做身份认证？**

身份认证是全球客户信息调查（KYC）合规标准的要求。完成 Bitget 身份认证有助于我们管理风险、防范欺诈，并保障您的账户和资产安全。

**2.身份认证与我使用 Bitget 服务有什么关系？**

自2024年1月1日起，使用加密货币充值和交易等核心服务均要求身份认证。您必须完成身份认证才能继续使用这些功能。

**3.完成身份认证后，我每天的提现额度是多少？**

身份认证通过后，您的每日提现额度即根据您的 VIP 等级提升：

| VIP 等级 | 24小时提现限额 |
| --- | --- |
| VIP0 | 3,000,000 USDT |
| VIP1 | 4,000,000 USDT |
| VIP2 | 6,000,000 USDT |
| VIP3 | 8,000,000 USDT |
| VIP4 | 10,000,000 USDT |
| VIP5 | 12,000,000 USDT |
| VIP6 | 15,000,000 USDT |
| VIP7 | 20,000,000 USDT |

请访问我们的**[官方费率页面](https://www.bitget.com/zh-CN/fee)**查看最新额度及费率详情。

**4.完成 Bitget 身份认证有什么好处？**

完成身份认证后，您可以：

- 使用加密货币和法币充值
- 使用 C2C 交易服务
- 启用链上提现功能
- 增强账户安全性

**5.身份认证审核过程需要多长时间？**

身份认证审核通常需要 60 分钟。审核完成后，您将通过电子邮件或账户总览页收到通知。如果您的身份认证处理时间超过 60 分钟，请[提交工单](https://www.bitget.com/zh-CN/ticket/ticket-form?ticketId=2061678148075433984)并提供您的用户 ID（UID）。我们的团队将协助您尽快解决问题。

**6.如何查看个人身份认证申请的状态？**

前往**用户中心**并选择“**[身份认证](https://www.bitget.com/zh-CN/verified/enter)**”。您的申请状态将显示在页面顶部。

**7.我可以在多个 Bitget 账户中使用同一个 ID 吗？**

不能，每位用户仅可认证一个 Bitget 账户。在多个账户中提交同一 ID 将导致身份认证被拒。

**8.我可以在子账户中完成身份认证吗？**

不行，身份认证只能在主账户上完成。认证通过后，您的子账户将自动继承相同的访问权限。

**9.我可以在多个 Bitget 账户上完成身份认证吗？**

不能，每位用户仅可认证一个 Bitget 账户。如果您需要多个交易账户，我们建议您在已完成身份认证的主账户下开设子账户。

**10.我一天可以尝试几次个人身份认证？**

24小时内，您最多可尝试5次身份认证。如果所有尝试均失败，您需要再等待24小时后再尝试。

**11.完成认证后，我还能更改姓名吗？**

如果您的身份认证信息与法定姓名不一致，您可以在[个人认证页面](https://www.bitget.com/zh-CN/verified/enter)使用**“编辑姓名”**功能进行更改。请提供正确的姓名并上传所需的证明文件。

**12.我可以删除身份认证信息吗？**

您无法删除身份认证信息，以下情况除外：

- 您的账户已被删除。
- 您改变了国籍（如移民）。
- 您无法使用 SEPA 账户进行法币充提。请确保 SEPA 账户和 Bitget 账户身份信息相同。

**注意：**如果用户滥用身份认证信息，并导致可能出现的账户纠纷或与欺诈有关的活动，用户须自行承担资金损失。

**13.重置身份认证后，我的账户会受到限制吗？**

会，您的账户将暂时变为未认证状态。您将无法充值或交易，直至新的认证完成。

**14.重置过程需要多长时间？**

重置过程通常需要1个工作日，具体时间取决于验证队列的处理情况。

**15.我如何知道我的身份认证申请是否被拒？**

您将收到 Bitget 账户内通知或电子邮件，其中将说明拒绝原因。

**16.如果我的身份认证申请被拒，我该怎么办？**

核实拒绝原因，解决问题（如上传更清晰的图片或提供有效文件），然后重新提交申请。
