# 如何在 Bitget 通过 SWIFT 充值与提现美元（USD）？

*[预计阅读时间：3 分钟]*

本文将介绍已完成身份认证的用户，如何通过 SWIFT 转账方式在 Bitget 进行美元（USD）的法币入金（充值）与出金（提现）。此流程要求用户完成身份认证（KYC/KYB）并使用同名银行账户。

**不支持的国家和地区**受合规限制，以下国家和地区不支持美元 SWIFT 转账：**阿富汗、白俄罗斯、古巴、中非共和国、俄罗斯、朝鲜、伊朗、缅甸、刚果民主共和国、利比亚、尼加拉瓜、南苏丹、叙利亚、乌克兰、委内瑞拉、巴勒斯坦（包含加沙地带）**

- 选择 USD（美元），并将 银行充值 (Bank deposit) 作为您的入金方式。
- 完成身份认证：
- KYC 用户：提供您的居住地址并同意用户条款。

- KYB 用户：同意机构账户条款。在某些情况下，KYB 用户可能还需要提供英文版的注册办公地址和公司地址，特别是当最初提交的 KYB 认证资料为非英文语言时。

![如何在 Bitget 通过 SWIFT 充值与提现美元（USD）？](https://img.bgstatic.com/multiLang/web/ca1e189e331c6596cd94d6291855092a.png)

## 如何通过 SWIFT 充值（入金）美元？

## 第一步：发起美元充值

1. 前往 快捷买币 (Buy Crypto) > 银行充值 (Bank deposit)。

![如何在 Bitget 通过 SWIFT 充值与提现美元（USD）？](https://img.bgstatic.com/multiLang/web/f6d2749c01202f957d6a4d38ec83db1d.png)

2. 选择 USD 作为币种，并选择 SWIFT 作为支付方式。

![如何在 Bitget 通过 SWIFT 充值与提现美元（USD）？](https://img.bgstatic.com/multiLang/web/94a8eee2fc56db6f5d70aa3a3cd69d9a.png)

3. 点击下一步 (Next)，输入您想要充值的金额。

## 第二步：通过银行转账

1. 核对 Bitget 提供的银行账户信息。

2. 请务必使用与您的 Bitget 账户同名的银行账户发起 SWIFT 转账。

3. 来自第三方账户或姓名不符的转账将被拒绝。

4. 退款可能需要较长时间，并可能产生手续费。

5. 您的资金将在 0至2个工作日 内到账。

美元 SWIFT 转账重要注意事项

## KYC 要求

- KYC 用户在开户过程中必须提供完整的居住地址。

## 同名账户政策充值

- **充值：**
  - 仅接受与您的 Bitget 账户法定姓名一致的银行账户发起的转账。
- **提现：**
  - 部分用户只能提取资金至之前成功充值过的银行账户。
  - 其他用户在提现前可能需要手动绑定提现银行账户，具体取决于账户状态和可用通道。
  - 添加或绑定提现账户可能需要按要求提交额外信息。
- 提现账户要求
  - 只能向与您个人档案关联的银行账户提现。
  - 银行账户姓名必须与您在 Bitget 认证的姓名一致。
  - 成功充值并不意味着自动绑定该银行账户用于提现。
  - 新的银行账户审核可能需要 1至3个工作日。

## 处理时间

- 充值和提现通常都需要 1至2个工作日 才能完成。

## 姓名不符处理

- 如果页面显示的收款人姓名与您的认证信息不符，请在提交请求前联系客服支持。

## 常见问题 (FAQs)

**1. 谁能在 Bitget 使用 SWIFT 转账进行美元的入金和出金？**

只有完成身份认证（KYC 或 KYB）并通过法币服务商审核的用户，才能使用 SWIFT 转账。

**2. 通过 SWIFT 进行美元充值和提现需要多长时间？**

充值和提现的处理时间通常为 0至2个工作日。

**3. 我可以使用第三方银行账户充值美元吗？**

Bitget 仅接受与您已认证的 Bitget 账户法定姓名一致的银行账户充值。第三方转账将被拒绝。

**4. 我可以通过 SWIFT 将美元提现到新的银行账户吗？**

提现仅支持之前已通过 SWIFT 成功充值过美元的银行账户。

**5. 如果我尝试使用姓名不符的账户充值会怎样？**

该笔转账将被拒绝，退款可能需要较长时间，并可能产生手续费。

**6. SWIFT 美元充值和提现需要手续费吗？**

通过 SWIFT 充值免手续费。关于提现手续费，请查阅相关费率文章了解详情。

**7. 为什么我之前保存的银行账户没有显示？**

受近期的系统或通道升级影响，之前保存的银行账户可能未被迁移。因此，您早期的银行账户可能不再显示在提现页面。请在提交提现前重新添加该银行账户。

**8. 为什么我无法提现到以前用过的银行账户？**

提现仅支持关联至您个人档案、且与您实名认证姓名一致的银行账户。

**9. 成功的美元充值会自动绑定我的银行账户用于提现吗？**

否。一次成功的充值并不会自动绑定该银行账户用于法币提现。

**10. 审核一个新的提现银行账户需要多长时间？**

新银行账户的审核通常需要 1至3个工作日。

**11. 添加新账户需要提供哪些银行信息？**

您需要提供账号或 IBAN、SWIFT 或 BIC 代码、银行名称以及银行所在城市。
