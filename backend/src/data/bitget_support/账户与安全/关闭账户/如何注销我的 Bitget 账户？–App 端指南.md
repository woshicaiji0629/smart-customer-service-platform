# 如何注销我的 Bitget 账户？–App 端指南

*【预计阅读时间：3分钟】*

本文将介绍如何通过手机 App 注销 Bitget 账户。请注意，注销账户操作不可逆转。确保提取所有资金并完成待处理交易后再继续操作。

## 开始前注意事项

- **账户永久注销**：账户一旦注销，将无法恢复，所有相关数据将被永久删除。
- **重新注册等待期：**注销后，您必须等待30天才能使用相同的手机号码、邮箱地址或身份证注册新账户。
- **平仓或强平所有仓位**：确保所有未平仓位全部平仓或强平。
- **提取所有资金**：确认您已清空所有账户且无待处理的提现。

## 如何通过 Bitget App 注销我的 Bitget 账户？

### 第1步：前往个人资料设置

1. 在 Bitget App 首页，点击左上角的菜单图标。

2. 在菜单屏幕上，点击顶部的个人资料栏进入个人资料设置。

3. 选择**“注销账户”**继续操作。

![如何注销我的 Bitget 账户？–App 端指南](https://img.bgstatic.com/multiLang/web/e021eeb5e85313a7f56fcf8da73c165b.png)

### 第2步：确认账户注销

1. 仔细查看提示内容。请确保已关闭所有仓位、提取所有资金，并清空您的 Web3 钱包。账户注销后，资产将无法恢复。

2. 勾选**“我已确认 Web3 钱包无可用资产”。**

3. 如果您希望继续，请点击**“注销账户”**。

### 第3步：完成账户注销

1. 请仔细阅读风险提示条款，勾选复选框以确认您已了解注销账户的后果，然后点击**“下一步”**以确认您的操作。

2. 您可能需要完成验证步骤，例如输入账户密码并进行安全验证。

3. 一旦提交请求，审核将在24小时内完成，您的 Bitget 账户将永久停用。

## 重要提示

此操作**不可逆转**。请确保在继续操作前已提取账户余额并关闭所有进行中的交易。

注销账户将导致以下后果：

- 所有交易和登录功能将停用。
- 与您账户关联的所有 API 私钥将停用。
- 所有已授权的设备将与您的账户解除绑定。
- 任何已验证的身份信息将被永久删除。
- 您的账户将不再接收任何奖励或奖金。
- 如果您之前使用 Bitget 的银行转账功能开设过银行账户，注销账户后，您将无法通过同一身份证关联的新账户再次使用该功能。

## 常见问题

**1.我的 Bitget 账户注销后还可以恢复吗？**

不能，账户一旦注销就无法恢复。您需要创建一个新账户才能再次使用 Bitget。

**2.注销账户后，我的个人数据会发生什么变化？**

您的个人数据将根据 Bitget 的隐私政策和适用法规被永久注销。

**3.如果账户中有资金或未完成的交易，我可以注销账户吗？**

不可以，您必须确保提取所有资金、完成所有交易并关闭所有未赎回申购（例如简单赚币、质押）后，才能注销账户。

**4.注销后我还能访问交易记录和身份信息吗？**

不能，所有交易记录和已验证的身份信息将被永久删除，账户停用后将无法访问。

**5.我的 API密钥和设备授权在注销后会发生什么变化？**

所有 API 密钥和链接到您账户的授权设备都将自动停用。

**6.我可以在注销旧账户后立即注册新账户吗？**

不可以。注销 Bitget 账户后，您必须等待**30天**才能使用相同的手机号码、邮箱地址或身份证注册新账户。请确保在注销之前考虑清楚。
