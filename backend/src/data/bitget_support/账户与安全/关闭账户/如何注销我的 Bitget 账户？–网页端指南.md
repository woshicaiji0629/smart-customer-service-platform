# 如何注销我的 Bitget 账户？–网页端指南

*【预计阅读时间：3分钟】*

本文介绍了如何通过网页端注销 Bitget 账户。请注意，注销账户操作不可逆转。确保提取所有资金并完成待处理交易后再继续操作。

## 开始前注意事项

- **账户永久注销**：账户一旦注销，将无法恢复，所有相关数据将被永久删除。
- **重新注册等待期：**注销后，您必须等待30天才能使用相同的手机号码、邮箱地址或身份证注册新账户。
- **平仓或强平所有仓位**：确保所有未平仓位全部平仓或强平。
- **提取所有资金**：确认您已清空所有账户且无待处理的提现。

## 如何在 Bitget 网页端注销我的 Bitget 账户？

### 第1步：前往个人资料页面

1. 点击页面右上角的个人资料图标，进入**个人资料**页面。

![如何注销我的 Bitget 账户？–网页端指南](https://img.bgstatic.com/multiLang/web/7f822f41d62d833a9abf025ede9aed3e.png)

### 第2步：访问我的个人资料

1. 在总览页中，点击**个人资料**。

2. 滚动到页面底部并点击[“注销账户”](https://www.bitget.com/zh-CN/account/setting/cancellation)。

![如何注销我的 Bitget 账户？–网页端指南](https://img.bgstatic.com/multiLang/web/935f62bc60ea127da891ccc6641cd123.png)

![如何注销我的 Bitget 账户？–网页端指南](https://img.bgstatic.com/multiLang/web/f9505b6b2473e1a4bfce4248eafef1c4.png)

### 第3步：确认账户注销

1. 阅读提示信息，勾选所需的确认框以确认您的操作。

2. 点击**“注销账户”**按钮以继续。

![如何注销我的 Bitget 账户？–网页端指南](https://img.bgstatic.com/multiLang/web/7ce9709e1e0b22a16867a9d1dcb5911e.png)

### 第4步：完成账户注销

1. 在确认并勾选**“我已知晓注销账户的后果”**后，点击**“注销账户”**以继续操作。

2. 您可能需要完成验证步骤，例如输入账户密码并进行安全验证。

3. 一旦提交请求，审核将在24小时内完成，您的 Bitget 账户将永久停用。

## 重要提示

此操作不可逆转。请确保在继续操作前已提取账户余额并关闭所有进行中的交易。注销账户将导致以下后果：

- 所有交易和登录功能将停用。
- 与您账户关联的所有 API 私钥将停用。
- 所有已授权的设备将与您的账户解除绑定。
- 任何已验证的身份信息将被永久删除。
- 您的账户将不再接收任何奖励或奖金。
- 如果您之前使用 Bitget 的银行转账功能开设过银行账户，注销账户后，您将无法通过同一身份证关联的新账户再次使用该功能。

## 常见问题

**1.我的 Bitget 账户注销后还可以恢复吗？**

不能，账户一旦注销就无法恢复。您需要创建一个新账户才能再次使用 Bitget。

**2.注销账户后，我的个人数据会发生什么变化？**

您的个人数据将根据 Bitget 的隐私政策和适用法规被永久注销。

**3.如果账户中有资金或未完成的交易，我可以注销账户吗？**

不可以，您必须确保提取所有资金、完成所有交易并关闭所有未赎回申购（例如简单赚币、质押）后，才能注销账户。

**4.注销后我还能访问交易记录和身份信息吗？**

不能，所有交易记录和已验证的身份信息将被永久删除，账户停用后将无法访问。

**5.我的 API密钥和设备授权在注销后会发生什么变化？**

所有 API 密钥和链接到您账户的授权设备都将自动停用。

**6.我可以在注销旧账户后立即注册新账户吗？**

不可以。注销 Bitget 账户后，您必须等待**30天**才能使用相同的手机号码、邮箱地址或身份证注册新账户。请确保在注销之前考虑清楚。
