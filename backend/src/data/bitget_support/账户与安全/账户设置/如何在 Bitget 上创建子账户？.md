# 如何在 Bitget 上创建子账户？

*【预计阅读时间：3分钟】*

本指南将指导您如何在 Bitget 上创建**子账户**。子账户功能允许您在一个主账户下管理多个交易子账户，从而优化风险管理、资金分配和团队协作。

## 什么是子账户？

**子账户**是 Bitget 主账户下的二级账户。它独立运行，但共享主账户的安全设置。子账户可用于不同的交易策略、团队管理，或是拆分资金。

## 子账户类型

Bitget 提供三种子账户：

**1. 虚拟子账户。**无需单独的邮箱或密码，可从主账户切换进入。

**2. 普通子账户**。需要邮箱和密码，具有与主账户不同的安全设置。

**3. 托管交易子账户**。用于委托专业交易团队管理资金。

## 如何在 Bitget 网站上创建子账户？

### 第1步：访问子账户总览

1. 鼠标悬停在右上角的**个人资料图标**。

2. 从下拉菜单中选择**子账户**。

![如何在 Bitget 上创建子账户？](https://img.bgstatic.com/multiLang/web/fb7f42ac56022b0a4d1175eb3257e2b1.png)

### 第2步：选择子账户类型

1. **在子账户**页面，单击右上角的“**创建”**按钮。

![如何在 Bitget 上创建子账户？](https://img.bgstatic.com/multiLang/web/3625befcab97bc8afaee076f8bb726f9.png)

2. 选择子账户类型

![如何在 Bitget 上创建子账户？](https://img.bgstatic.com/multiLang/web/be7ce867215b9053a069390793b8a0a1.png)

3. 根据所选类型输入子账户详细信息

4. 如果要为自动交易生成 API Key，请切换到“**创建 API**”。

### 第3步：确认并创建

1. 点击“**创建”**完成创建过程。

## 管理子账户

创建子账户后，您可以：

- **切换账户。**使用主账户管理虚拟子账户。
- **划转资金。**在主账户和子账户之间即时划转资产。

## 常见问题

1. 我可以创建多少个子账户？

默认情况下，您最多可以创建**20个子账户**。

2. 子账户可以提现资金吗？

要提现，您必须先**将资金转入主账户**，然后**从主账户**进行提现。

3. 可以删除子账户吗？

是的，您可以删除子账户。删除前，请确保子账户所有仓位已平仓、所有余额已提现。请妥善管理账户中的剩余余额。删除后，账户中剩余的任何资产将被视为没收，无法找回。

4. 子账户是否有单独的登录凭证？

只有**普通子账户**需要单独的邮箱和密码。**虚拟子账户**没有单独的登录凭证。

5. 我可以使用子账户进行 API 交易吗？

是的，您可以为具有自定义权限的子账户生成**API Key**。
