# 如何使用二维码通过 Bitget 网页端登录

*【预计阅读时间：3分钟】*

本文将逐步介绍如何在网站上使用二维码登录 Bitget 账户，无需手动输入邮箱、密码或安全码，从而简化登录过程。

## 注意事项

- 要在 Bitget 网站上使用二维码登录，必须在您的设备上安装 Bitget App。
- 该二维码具有时效性，如果过期，请刷新页面。
- 使用二维码登录可实现快速访问，无需输入邮箱、密码或安全码。

## 如何使用二维码登录

### 第1步：前往 Bitget 登录页面

1. 进入 Bitget 网站的"[登录](https://www.bitget.com/zh-CN/login)"页面

2. 点击登录页面旁边显示的**二维码**。

![如何使用二维码通过 Bitget 网页端登录](https://img.bgstatic.com/multiLang/web/5c5c221a26036972b24ccc7ad2385720.png)

### 第2步：打开 Bitget App

1. 在手机上启动 Bitget App。

2. 点击 App 左上角的**用户个人资料图标**。

3. 然后，点击个人资料页面右上方的**二维码图标**。

![如何使用二维码通过 Bitget 网页端登录](https://img.bgstatic.com/multiLang/web/e2ec805eeb6868ff49625e32af8c142b.png)

### 第3步：扫描二维码

1. 使用 Bitget App 扫描二维码。

2. 点击“**授权登录”**，即可在网站上成功登录您的 Bitget 账户。

### 第4步：如果二维码过期，请刷新

1. 如果二维码过期，只需刷新页面即可生成新的二维码。

2. 按照同样的步骤扫描新的二维码并登录。

## 常见问题

**1.如果二维码过期了，该怎么办？**

刷新页面以生成新的二维码代码，然后再次尝试扫描。

**2.我可以在多个设备上使用二维码登录吗？**

可以，只要您拥有 Bitget App 和支持的浏览器，就可以在各种设备上使用二维码登录。

**3.扫描二维码后需要输入邮箱或密码吗？**

不用，扫描二维码即可登录，无需重新输入邮箱、密码或安全码。

**4.为什么我的手机无法扫描二维码？**

请确保您的 App 使用的是最新版本，且将摄像头对准二维码。

**5.如果我没有 App，可以使用二维码登录 Bitget 网站吗？**

目前二维码登录需要使用 Bitget App 进行验证。
