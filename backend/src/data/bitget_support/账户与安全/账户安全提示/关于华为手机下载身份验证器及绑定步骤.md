# 关于华为手机下载身份验证器及绑定步骤

1、 首先通过华为应用市场--搜索**Authenticator**进行下载安装如下图：

![关于华为手机下载身份验证器及绑定步骤](https://img.bgstatic.com/multiLang/web/4d8defceada04c664da338288ebf8bd2.png)

需要注意：开发者为 （微软中国）有限公司

2、 打开您的手机app--左上角点击九宫格图标--进入个人资料页面点击UID后方的“ > ”--安全中心--谷歌验证。如下图：

![关于华为手机下载身份验证器及绑定步骤](https://img.bgstatic.com/multiLang/web/3a2b3f93c299d0d4032af2ffdb6d783e.png)

3、 打开**Authenticator**app--已验证ID--扫描QR码。如下图：

![关于华为手机下载身份验证器及绑定步骤](https://img.bgstatic.com/multiLang/web/047774b6ba15d80609014751e4aac72f.png)

4、 点击扫描QR码之后，扫描绑定谷歌验证器界面的二维码，在**Authenticator**会生成一个六位数的一次性密码代码--把6位数代码输入到绑定谷歌验证器界面的谷歌验证码位置--点击下一步按照步骤操作即可绑定成功。如下图：

![关于华为手机下载身份验证器及绑定步骤](https://img.bgstatic.com/multiLang/web/28445d9f387bc538f374965b58a286c0.png)

5、 谷歌验证器绑定成功，之后登录或者提币等操作都需要使用谷歌验证码。
