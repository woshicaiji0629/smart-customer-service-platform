# 如何为我的 Bitget 账户设置 PIN 码？- App 端指南

*【预计阅读时间：3分钟】*

本指南介绍如何在手机 App 上为您的 Bitget 账户设置 PIN 码。设置 PIN 码可以为特定操作增加一层保护，从而提高账户安全性。

## 为什么要设置 PIN 码？

PIN 码的作用：

- **增强安全性**：防止未经授权的访问。
- **易于使用**：方法简单，可保障账户安全。

## 如何在 Bitget App 上设置 PIN 码？

### 第1步：前往安全设置

1. 登录并点击左上角的**账户中心图标**。

2. 点击您的**个人资料部分**，进入**用户中心。**

3. 从菜单中选择“**安全中心**”。

![如何为我的 Bitget 账户设置 PIN 码？- App 端指南](https://img.bgstatic.com/multiLang/web/67785affe5fdea754c978e7b7f56dc8d.png)

### 第2步：找到 PIN 码部分

1. 在**安全中心**页面向下滚动，找到“**PIN 码**”选项。

2. 点击“**PIN 码**”开始设置。

### 第3步：创建 PIN 码

1. 输入**6位数 PIN 码**。确保该数字组合只用于该 PIN 码，且安全保存。您可以将 PIN 码写下来，并存放在一个只有您能接触到的安全场所。

### 第4步：进行安全验证

为确保安全，您需要完成安全验证：

- **邮箱验证**：输入发送到您注册邮箱地址的验证码。
- **谷歌验证**：输入验证器生成的六位数代码。
- **短信验证**：输入发送到您注册手机号的验证码。

### 第5步：确认设置

输入验证码后点击“**确认**”。您的 PIN 码将被激活。

## PIN 码安全小技巧

- 避免使用“000000”或出生日期等常规数字。
- 定期更新 PIN 码以提高安全性。
- 请勿与他人共享您的 PIN 码。

## 常见问题

**1. 设置后可以更改 PIN 码吗？**

是的，您可以随时前往**安全中心**，选择“**修改 PIN 码**”进行更改。确认更改时需要进行安全验证。

**2. 如果我忘记 PIN 码了怎么办？**

如果忘记 PIN 码了，请进入**安全中心**并选择**“重置 PIN 码”**。您需要使用邮箱、谷歌验证器或短信进行安全验证。

**3. PIN 码功能是强制性的吗？**

不，PIN 码是可选项，但我们建议启用 PIN 码以增强安全性。

**4. 我可以给多个账户设置相同的 PIN 码吗？**

出于安全考虑，最好为每个账户设置各不相同的唯一 PIN 码。

**5. 在 Bitget 上的每个操作都需要 PIN 码吗？**

不需要，只有特定操作（如提现或敏感账户变更）才需要 PIN 码。
