# 如何录制验证视频？

*【预计阅读时间：5分钟】*

本指南可帮助您录制视频，用于**重置您的安全项**、**完成身份认证**、**验证提现相关记录**或**其他相关应用**。确保您的视频清晰地拍摄到所有必要的细节，以加快审核过程。

## 录制前，请确保：

•**光线充分，背景整洁**，以便内容清晰可见。

• 使用**带有摄像头的设备**（例如智能手机）。

• 准备好您的**有效身份证件**。

## 如何录制验证视频

请在**一段连续的视频**中完成以下步骤。

## 第1部分：拍摄身份证证件正反面

1. 调整摄像头位置，清晰拍摄证件**正面**。

2. 保持稳定几秒钟。

3. 翻转证件，拍摄证件**背面**并保持几秒钟。

## 第2部分：手持身份证件，并确保拍摄到脸部

1. 切换到**自拍模式**或使用前置摄像头。

2. 将身份证**放在脸旁**，确保身份证及脸部都清晰可见。

3. 保持身份证稳定，**不要遮住脸部**。

## 场景：提现记录验证

对于涉及提现记录验证的情况，视频中应包含所有**相关的提现详情。**根据您的提现类型，所有步骤必须录制在**一段连续视频中**：

## 提现类型1——加密货币交易所 App

1. 打开提现平台的 App。

2. 显示登录过程。

3. 找到具体的提现记录。

4. 显示**交易 ID （TXID）**。

## 提现类型2——Web3 钱包

1. 打开钱包。

2. 显示钱包列表。

3. 选择相应的钱包。

4. 显示**钱包地址**。

5. 找到具体的提现记录。

6. 显示**交易ID**。

## 提现类型3——银行转账

1. 登录您的银行 App，并展示登录过程。

2. 找到相关的充值交易记录。

3. 显示**日期、时间、金额和币种**。

**注意：**登录过程可以通过**输入密码**、**面容 ID**、**指纹**或**通行密钥**的方式来展示。

## 常见的验证视频问题和解决方案

1. **录制时无法切换摄像头**

部分设备不支持在录制过程中切换前后摄像头。尝试以下解决方案：

• 使用**屏幕录制**，然后打开相机。

• 拍摄**身份证的正面和背面**，然后拍摄您**手持身份证时的脸部**。

• 在提交之前，**将两部分内容剪切成一段视频。**

2. **录制的视频不连续**

我们只接受**一段**包含所有所需内容的完整视频。为避免重复录制，请按照以下方式进行：

• 提前计划好录制顺序，确保录制顺利完成。

3. **视频文件过大，无法发送**

如果视频文件太大，无法通过电子邮件发送，可使用**云存储服务**（如 Google Drive）。

**• 将视频上传**到云端并生成共享链接。

• 确保**共享权限已开启**，以便客服部门访问链接。

## 常见问题

**1.我为什么需要录制验证视频？**

在重置验证方式、完成身份认证或验证提现记录时，您需要提交一段视频以确认您的身份。这样可以确保您的账户和交易得到安全验证。

**2.我可以分段提交视频吗？**

不可以，您只能提交一段连续的视频。请提前计划好录制顺序，确保录制顺利完成。

**3.视频应该多长？**

没有严格的时间限制，但必须确保录制中所有要求的细节都清晰可见。

**4.如果我的视频被拒绝，我该怎么办？**

如果视频被拒，可能是因为身份证件内容或脸部模糊不清，或缺少所需的内容。请重新录制视频，以确保所有必需内容均已涵盖且画面清晰。

**5.我在验证视频中应该按照哪种提现方式进行操作？**

请记录与您的提现方式相符的步骤：加密货币交易所 App、Web3 钱包或银行转账。请在一段连续的视频中记录该类型的所有操作步骤。

**6.我该如何在验证视频中展示登录过程？**

登录过程可以通过输入密码、面容 ID、指纹或通行密钥的方式来展示。
