# 如何为我的 Bitget 账户设置通行密钥？- App 端指南

*【预计阅读时间：3分钟】*

本指南将帮助您在 App 端为您的 Bitget 账户设置通行密钥。通行密钥可提供更高的安全性和更流畅的登录体验。按照以下步骤即可快速、轻松地启用该功能。

**什么是通行密钥？通行密钥**

是一种安全、无需密码的身份认证方法，采用基于设备的现代化验证方式，取代传统的密码。无需输入密码或验证码，通过

**指纹**

、

**人脸识别**

、

**设备 PIN 码**

或 YubiKey 等

**安全设备**

即可确认您的身份。

- 通行密钥可实现更快、更安全的登录体验——非常适合登录网站或应用、执行提现或更改设置等敏感操作。
- 这项技术得到了**苹果**、**谷歌**、**微软**等主流公司的支持，由**FIDO 联盟**在全球推广，并已集成到大多数现代浏览器和设备中。

**通行密钥的主要特性1.无需存储或记忆密码**

- 您无需创建或记忆密码。Bitget 也不会在其服务器上存储任何密码，有效降低密码泄露或被黑客攻击的风险。

**2.公钥-私钥加密技术保护**

通行密钥采用

**非对称加密技术**

。

- Bitget 会存储您的**公钥**，用于验证登录。
- 您的**私钥**则安全保存在您的个人设备（如手机、笔记本电脑或 YubiKey）上。
- 私钥**永远不会离开您的设备**，即使在登录期间也是如此，增强您的账户抵御服务器端攻击的能力。

**3.防范网络钓鱼攻击**

通行密钥

**与特定域绑定**

，只能在官方 Bitget 域内使用。

- 如果您访问的是仿冒或钓鱼网站，通行密钥将无法激活。
- 这可防止攻击者诱骗您输入凭据，不会像电子邮件、短信或验证码那样被拦截或滥用。

**4.内置多因子验证（MFA）**

通行密钥会自动应用两层保护：

- **受信任设备**——您必须使用个人拥有的设备，例如您的智能手机或电脑。
- **个人解锁方式——**为确认是您本人使用，设备会要求您提供只有您才能提供的信息，例如指纹、面容 ID 或锁屏 PIN 码。

## 如何为您的 Bitget 账户设置通行密钥 ？

### 第1步：前往用户中心

1. 打开**Bitget App**，登录您的账户。

2. 点击左上角的**九宫格图标**。

3. 点击您的**个人资料部分**，进入**用户中心。**

![如何为我的 Bitget 账户设置通行密钥？- App 端指南](https://img.bgstatic.com/multiLang/web/16d6b7ced1dd6f859ee7e9ba33be0de3.png)

### 第2步：开始设置通行密钥

1. 从菜单中选择“**安全中心**”。

2. 向下滑动找到**通行密钥**部分，点击“**添加通行密钥**”设置。

![如何为我的 Bitget 账户设置通行密钥？- App 端指南](https://img.bgstatic.com/multiLang/web/31d7e365798f3bd01f314452af419a19.png)

### 第3步：验证和保存

1. 完成所需的**安全验证**（如输入 2FA 验证码）。

2. 按照屏幕上的说明，使用设备上的生物识别系统验证身份。

3. 设置完成后，您的通行密钥即可注册成功。

### 第4步：测试通行密钥

1. 在 App 中退出您的 Bitget 账户。

2. 选择**使用通行密钥**验证，尝试重新登录。

3. 完成生物识别或 PIN 码验证以访问您的账户。

## 常见问题

**1. 我可以在多个设备上使用通行密钥吗？**

可以，您在任何支持生物识别验证的设备上都可以设置通行密钥。

**2. 如果我的设备丢失了怎么办？**

您可以使用电子邮件/电话号码和密码登录账户，前往**安全中心**停用已丢失设备的通行密钥。

**3. 我可以同时使用 2FA 和通行密钥吗？**

可以，将通行密钥与双因子身份验证（2FA）结合使用还能提高安全性。

**4. 如何停用通行密钥？**

前往 Bitget App 的**安全中心**，在**通行密钥**部分点击“**移除通行密钥**”即可将其停用。

**5. 如果没有生物识别身份验证，还可以使用通行密钥吗？**

可以，如果您的设备不支持生物识别技术，您可以设置并使用安全 PIN 码。
