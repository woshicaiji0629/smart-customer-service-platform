# 如何修改/重置/停用 Bitget 账户的 PIN 码？- App 端指南

*【预计阅读时间：4分钟】*

本指南介绍如何在 Bitget App 端修改、重置或停用您 Bitget 账户的 PIN码。这些功能帮助您有效管理 PIN 码，从而提高安全性和便利性。

## 为什么要修改、重置或停用 PIN 码？

- **修改**：定期更新 PIN 码可提高账户安全性。
- **重置**：如果忘记了 PIN 码，可重置以恢复访问您的账户。
- **停用**：如果不再需要 PIN 码功能，请清除设置。

## 如何修改、重置或停用您的 PIN 码？

### 第1步：前往安全中心

1. 登录并点击左上角的**账户中心图标**。

2. 点击您的**个人资料部分**，进入**用户中心。**

3. 从菜单中选择“**安全中心**”。

![如何修改/重置/停用 Bitget 账户的 PIN 码？- App 端指南](https://img.bgstatic.com/multiLang/web/67785affe5fdea754c978e7b7f56dc8d.png)

### 第2步：找到 PIN 码部分

1. 在**安全中心**页面向下滚动，找到“**PIN 码**”选项。

2. 点击“**PIN 码**”选项，选择所需的操作：

- **修改 PIN 码**：更新 PIN 码。
- **重置 PIN 码**：如果忘记了当前的 PIN 码，可重新设置。
- **停用 PIN 码**：停用 PIN 码功能。

### 第3步：完成设置

1. 选择所需的操作后：

- **修改 PIN 码**：输入新的6位数字 PIN 码，提供当前 PIN 码并完成安全验证。
- **重置 PIN 码**：输入新的6位数字 PIN 码并完成安全验证。
- **停用 PIN 码**：输入当前 PIN 码并完成安全验证。

2. 点击“**确认**”完成操作。

## PIN 码管理小技巧

- 定期更新 PIN 码，可提高安全性。
- 避免使用“123456”这样的简单组合或您的出生年份等容易被猜到的 PIN 码。
- 妥善保管您的 PIN 码，不要与他人共享。

## 常见问题

**1. 忘记 PIN 码怎么办？**

使用安全中心的“**重置 PIN 码**”功能，完成安全验证即可设置新的 PIN 码。

**2. 如果忘记了当前 PIN 码，可以直接修改 PIN 码吗？**

不，您需要验证当前的 PIN 码才能修改。如果忘记了，请重置 PIN 码。

**3. 停用 PIN 码安全吗？**

停用 PIN 码就等于取消了一层安全保护。请确保已启用 2FA 等其他安全措施。

**4. 停用 PIN 码后还可以重新启用吗？**

可以，您可以随时前往安全中心，重新启用并设置新的 PIN 码。

**5. 我可以在没有安全验证的情况下重置 PIN 码吗？**

不可以，必须完成安全验证才能重置 PIN 码。这是为了确保只有授权用户才能进行更改。
