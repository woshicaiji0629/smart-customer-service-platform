# 如何在 Bitget 完成 RFI？

*【预计阅读时间：3分钟】*

Bitget 始终致力于保障用户资金安全并遵守全球的监管要求。如果您收到要求填写信息请求（RFI）的通知，本指南将为您提供相关分布指南。及时完成 RFI 有助于平台核实账户信息，排查异常活动，并确保平台交易安全。

**什么是 RFI？**

RFI（信息请求）是 Bitget 提出的一项正式请求，要求用户提供更多细节或文件，以便核实账户信息或应对特定的监管要求。

**您收到 RFI 的原因如下：**

- **账户风险评估**：如果账户活动表现异常或与正常模式不一致，可能会触发风险评估。
- **风险控制措施**：针对高风险交易、活动或来自未经验证地区的访问，启动风险控制。
- **合规需求：**为确保符合反洗钱（AML）及其他金融法规。

请放心，所有在此过程中收集的信息将严格保密，仅用于身份认证。更多详情，请参阅我们的[《使用条款》](https://www.bitget.com/zh-CN/support/articles/360014944032-terms-of-use)，特别是第5.3条（iii）–（vi）款。

**如何完成 RFI 流程？第1步：访问 RFI 请求**

- 请检查您的邮箱，查找来自 Bitget 的邮件，邮件主题为：**“【重要】更新您的账户信息以提升账户安全。”**打开邮件并按照指示完成操作。
- 您也可以通过 Bitget[官网](https://www.bitget.com/zh-CN)或 App 登录您的账户。如需完成 RFI，您将在总览页面收到通知或提示，指导您进行下一步操作。
- 单击【**立即开始】**按钮开始流程。

**第2步：查看 RFI 请求**

请仔细阅读指示，并注意所需的具体细节或文件。 常见的请求文件：

- 政府颁发的身份证明（如护照、驾照等）。
- 地址证明（如水电账单、银行对账单等）。
- 其他交易细节或补充说明。

**第3步：上传所需文件**

- 使用**【上传文件】**按钮提交所需文件。
- 确保文件**清晰且分辨率高**，**四角必须可见。第4步：提供补充信息**

- 如果有提示，完成必要表单或文本字段以提供相关信息（例如交易解释）。

**第5步：提交 RFI 回复**

- 请仔细检查所有信息和上传的文件是否准确无误且清晰。
- 点击**【提交】**按钮，将您的 RFI 回复发送给 Bitget 审核。

**常见问题1. Bitget 审核我的 RFI 需要多久？**

审核时间通常为1–3个工作日，具体时长根据情况复杂程度而定。如需额外信息，Bitget 会通过您的注册邮箱或账户总览联系您。

**2. 如果我没有收到 RFI 通知怎么办？**

请检查您的邮箱，查找来自**Bitget**的邮件，邮件主题为：**“【重要】更新您的账户信息以提升账户安全。”**该邮件包含了所有必要信息，并提供直接链接，您可以点击进入 RFI 流程。如果无法找到邮件，请查看垃圾邮件或垃圾箱。

**3. 如果我没有完成 RFI 怎么办？**

未完成 RFI 可能会导致账户访问受限，包括无法进行交易、提现或充值，直到提供所需信息为止。

**4. 我如何查看我的 RFI 状态？**

你可以在

[RFI 页面](https://www.bitget.com/zh-CN/ticket/rfi)

查看你的 RFI 问卷状态。
