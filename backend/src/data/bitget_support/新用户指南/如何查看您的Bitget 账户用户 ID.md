# 如何查看您的Bitget 账户用户 ID

*【预计阅读时间：3分钟】*

本文将解释什么是**Bitget 账户用户 ID**，并展示如何在 Bitget 网站和 App 上查找并复制。

## 什么是 Bitget 账户用户 ID？

您的用户 ID**（用户身份识别号码）**是注册 Bitget 账户时自动生成的唯一标识。我们使用您的用户 ID 来识别您的账户，以便提供客户服务和其他与账户相关的服务。

## 重要提示：

•您的用户 ID**不是**您的注册手机号码或邮箱地址。

•每个 Bitget 账户都有**唯一一个用户 ID**。

## 如何查看您的 Bitget 账户用户 ID

## 网页端：

1. 登录 Bitget 网站。

2. 将鼠标悬停在右上角的**“个人资料”**图标上。

3. 个人资料页面将显示您的用户 ID。

4. 点击复制您的用户 ID。

![如何查看您的Bitget 账户用户 ID](https://img.bgstatic.com/multiLang/web/c1e6ffafe6ad6606944cb4db1ef76f94.png)

5. 您也可以进入**[总览](https://www.bitget.com/zh-CN/dashboard/profile)**页面查看并复制您的用户 ID。

![如何查看您的Bitget 账户用户 ID](https://img.bgstatic.com/multiLang/web/cb6682b64bb2baebb03e7014065479af.png)

## App 端：

1. 打开 Bitget App 并登录。

2. 点击首页左上角的**“个人资料”**图标。

3. 点击**个人资料**。

4. 您的用户 ID 将在屏幕上显示。

5. 点击复制您的用户 ID。

![如何查看您的Bitget 账户用户 ID](https://img.bgstatic.com/multiLang/web/d8782ca067f6c7e1c2819f8733bf5d17.png)

## 常见问题

**1.什么是 Bitget 账户用户 ID？**

用户 ID 是您注册 Bitget 账户时自动生成的唯一身份识别号码。

**2.我的用户 ID 有什么用途？**

用户 ID 用于在客服和其他账户相关服务中识别您的账户。

**3.我的用户 ID 和我的邮箱或手机号码一样吗？**

不一样，您的用户 ID 与您的注册邮箱地址和手机号码不同。

**4.我可以更改我的 Bitget 账户用户 ID 吗？**

不可以，您的用户 ID 是自动生成的，无法更改。
