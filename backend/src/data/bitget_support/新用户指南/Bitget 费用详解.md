# Bitget 费用详解

*【预计阅读时间：4分钟】*

本文全面介绍了 Bitget 的所有费用，包括交易、充值、提现和其他平台相关费用。了解这些费用有助于您有效管理资金，优化交易体验。

## 交易手续费

Bitget 根据您的**VIP 等级**以及您在订单簿中是**挂单方**还是**吃单方**收取交易手续费。

**1. 现货手续费**

- **挂单费率：**0.1%（使用**BGB**支付费用时可享受**0.08%**的折扣费率）
- **吃单费率：**0.1%（使用**BGB 支付费用**时可享受**0.08%**的折扣费率）

**2. 合约手续费**

- **挂单费率：**0.02%
- **吃单费率：**0.06%
- 您可以通过持有**BGB**或增加交易量来减少费用。

**3. 杠杆利率**

- 杠杆利率因**代币**和**杠杆类型**而异。VIP 用户可享受**分级利率折扣**，随着等级的提高，借贷成本也随之降低。
- 您可以查看**[费率标准](https://www.bitget.com/zh-CN/fee)**页面了解详情。

**4. VIP 折扣**

- **VIP 用户**可享受梯级费用折扣。
- 您可以根据**资产、BGB 持有量和交易量**提升**VIP 等级**，以享受更低的费用。
- 查看**[VIP 费率标准](https://www.bitget.com/zh-CN/fee?tab=1)**了解详细信息。

## 充提手续费

**1. 充值手续费**

- **加密货币充值：**加密货币充值在 Bitget 上**免费进行**，但需要支付**网络费用**，**发送平台或交易所**可能会收取**提现手续费**。
- **法币充值：**视支付方式和提供商**而定**（可能需要支付第三方费用）。

**2. 提现手续费**

- Bitget 根据**所选加密货币和网络**收取提现费用。这些费用包括区块链交易成本，可能会有所波动。提现前，您可以在**提现页面**上查看具体费用。

有关**充值和提现费用的完整情况**，请查看**[费率标准](https://www.bitget.com/zh-CN/fee)**。

## 其他费用

**1. 合约资金费用**

- 在永续期货合约中**每8小时**收取一次。
- 当**资金利率为正数**时，持有**多头仓位的**交易者向持有**空头仓位**的交易者支付资金。当出现**负数**时，**空头仓位**交易者会向**多头仓位**交易者支付费用。
- **计算公式：资金费用 = 仓位价值 x 资金费率**

如需了解实时资金费率，请[点击此处](https://www.bitget.com/zh-CN/futures/introduction/funding-rate)。

**2. 跟单交易费用**

- 跟单者向**交易专家**支付的**分润**（通常为**10%-20%**），具体取决于交易专家的设置。
- 当交易专家平仓时，如果跟单者的交易产生利润并满足**分润标准**，利润的一定比例会**自动**从跟单者的账户中**扣除**，并存入交易专家的账户。

**3. C2C 交易手续费**

- 在**Bitget C2C 市场**上，买卖双方**零费用**。
- 不过，根据您的**支付方式**或**银行政策**，可能需要支付额外费用。

**4. 兑换费用**

- **Bitget 闪兑**交易**不收取费用**。
- 根据市场情况，可能会有**少量价差**。

## 常见问题

1. Bitget 存在什么隐形费用吗？

不存在，所有费用都在 Bitget[费率标准总览](https://www.bitget.com/zh-CN/fee?tab=1)页面上以透明的方式展示。

2. 在 Bitget 上持有加密货币需要支付费用吗？

不用，平台不收取维护费用或托管费用。

3. 为什么提现手续费有所不同？

提现费用取决于**区块链网络拥堵**程度和**交易成本**。

4. 我可以获得交易手续费折扣吗？

可以，使用**BGB**或提高**VIP 等级**可以减少费用。

5. 是否有最低充值费用？

没有，充值可免费进行，但**第三方支付提供商**可能会收取费用。
