# 如何读懂盈亏分析？

*【预计阅读时间：3分钟】*

读懂**盈亏（PnL）分析**对于评估您的交易业绩至关重要。Bitget 的盈亏分析分为三个选项卡：**账户分析**、**币种分析**和**交易分析**，每个选项都能为您的交易活动提供独特的见解。

**什么是盈亏（PnL）？PnL 的全称是 Profit and Loss（盈利和亏损）**，是一种跟踪您在加密货币交易活动中赚了多少钱或亏了多少钱的方法。它将您的交易业绩分成三个简单的部分，帮助您了解自己的交易业绩：

**1. 已实现盈亏：**已完成交易的盈利或亏损。例如，如果您以85,000美元买入[比特币](https://www.bitget.com/zh-CN/price/bitcoin)，并以90,000美元卖出，扣除手续费，您的利润为5000美元。

**2. 未实现盈亏：**您持有币种的潜在盈利或亏损。例如，如果您以3000美元买入以太坊，现在价值3500美元，那么您的未实现盈亏为500美元。

**3. 总盈亏：**已实现盈亏和未实现盈亏之和，显示您的整体交易表现。

**如何查看 Bitget 网站上的盈亏分析？**

1. 前往**“资产”**，然后从选项中选择“**现货账户”**。

![如何读懂盈亏分析？](https://img.bgstatic.com/multiLang/web/7372dfa30e83a44349818aa690ccfe73.jpeg)

2. 选择**“盈亏分析”**

![如何读懂盈亏分析？](https://img.bgstatic.com/multiLang/web/44744ded9ebc2583e7d5c5b125cf06a1.jpeg)

3. 选择**“盈亏分析**”后，您将看到以下选项卡：

- **账户分析**
- **币种分析**
- **交易分析如何在 Bitget App 上查看盈亏分析？**

1. 点击底部导航栏上的“**资产”**。

2. **点击“今日”**，进入现货盈亏分析页面。

![如何读懂盈亏分析？](https://img.bgstatic.com/multiLang/web/3574d9e05de66e9ea1a0c1b709311111.jpg)

**账户分析**

“账户分析”选项卡汇总了您的总体盈亏和资产价值。

1. **“现货估值”**显示以 BTC 和等值美元计算的资产总值。

2. **“今日盈亏”**显示当日以美元计价的盈亏。

3. **“7日盈亏”和“30日盈亏”**提供过去7天和30天的累积盈亏数据，帮助您查看短期业绩。

4. **“自定义”**日期范围允许您选择特定日期来计算任何特定时期的盈亏总额。

**币种分析**

“币种分析”选项卡可让您深入了解投资组合中单个加密货币的表现。

1. **“总额”**显示所选加密货币的总持有量和当前价值。

2. **“交易持仓量（BTC）”**显示您当前持有的货币数量。

3. **保本价（USDT）**反映了在考虑持仓量和交易手续费变化的情况下，您可以在不产生亏损的情况下卖出资产的平均价格。当您进行新的交易或调整仓位时，该价格就会更新，帮助您跟踪回本所需的准确价格。

4. **成本价（USDT）**代表您资产的平均成本，包括初始仓位和后续购买的仓位。它将额外补仓的价格计算在内，让您清楚地了解随着时间推移的总投资成本。

5. **最新价（USDT）**显示所选加密货币的当前市场价格。

6. **“累计盈亏”**和**“累计盈亏率”**反映了特定币种的总盈亏和收益率。

**交易分析**

“交易分析”选项卡提供了特定币对的详细交易业绩明细。

1. **“基础币/报价币”**可让您选择币对（如[BTC/USDT](https://www.bitget.com/zh-CN/spot/BTCUSDT)）进行分析。

2. **“已实现盈亏”**显示所选币对已完成交易的盈亏。

3. **“未实现盈亏”**显示未平仓位与购买价格相比的潜在收益或损失。

4. **“实付手续费”**显示已支付的费用，您还可选择启用“BGB 折扣减免”获得折扣。

**常见问题1. 我能否为特定币对定制盈亏分析？**
是的，您可以在“**交易分析”**选项卡中使用“**基础币/报价币”**下拉菜单选择特定的交易对。

**2. 如何计算特定币种的收益率？**收益率在“币种分析”选项卡中自动计算，显示为收益占总投资的百分比。

**3. 为什么我的未实现盈亏显示为0？**
只有在所选币对中有未平仓位时，才会显示未实现盈亏。如果没有活跃交易，则显示0。

**4. 盈亏分析中如何计算交易手续费？**
交易手续费根据您的交易量和手续费等级扣除。您可以在“**交易分析”**选项卡的“**实付手续费”**部分查看详细明细。

**5. 我应该多久查看一次盈亏分析？**每天查看或在重大交易或市场变动后查看您的盈亏，随时了解您的最新交易业绩。
