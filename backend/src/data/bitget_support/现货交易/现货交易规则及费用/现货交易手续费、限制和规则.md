# 现货交易手续费、限制和规则

*【预计阅读时间：4分钟】*

本文概述了 Bitget 现货交易的交易手续费、限制和规则。了解这些关键要素将帮助您优化交易体验，在遵守 Bitget 平台政策的同时降低成本。我们还将介绍针对高级用户的 VIP 级费用结构。

**现货交易手续费概览**

Bitget 采用清晰明了的现货交易手续费结构，适合各种经验水平的交易者。采用“**挂单-吃单”模式**，挂单和吃单交易的**基准手续费均为0.1%：1. 挂单费：**

- 对增加市场流动性的订单收取手续费，如未匹配成功的限价单。
- **基准利率：**0.1%。

**2. 吃单费：**

- 对消除流动性的订单收取手续费，如市价单或立即匹配的限价单。
- **基准利率：**0.1%。

**如何获得交易手续费折扣？1. 使用 BGB 享受折扣：**

- 在进行现货或杠杆交易时，使用**BGB**扣除交易手续费，即可享受**20%的折扣**。
- 现货交易手续费将直接从您现货账户中的可用**BGB 余额**中扣除。

**2. VIP 费率标准**

- Bitget 为交易大户提供专属 VIP 费率标准，提供更具竞争力的费率。VIP 交易者可根据其交易量和 BGB 持有量，享受挂单和吃单手续费减免。有关 VIP 手续费收费等级及其相应折扣的详细信息，请查看Bitget 费率标准。

**现货交易限制**

Bitget 设置一定的交易限制，以维护公平高效的交易环境。

**1. 最小订单金额**

- 最小交易量因币对而异，以确保流动性和交易的顺利执行。

*例如：*

- **[BTC/USDT](https://www.bitget.com/zh-CN/spot/BTCUSDT)：**0.0001 BTC
- **[ETH/USDT](https://www.bitget.com/zh-CN/spot/ETHUSDT)：**0.001 ETH

**2. 最大订单金额**

- 最大订单量由每个币对的流动性和市场条件决定。
- 要查看特定币对的最大订单金额，请前往 Bitget 账户中的**现货交易**。

**3. 每日交易限额**

- 现货交易未设置**具体的每日交易限额**。

更多详情，请访问[现货交易规则](https://www.bitget.com/zh-CN/spot/introduction/spot-rules)。

**主要现货交易规则1. 账户要求**

- 用户必须完成**[身份认证](https://www.bitget.com/zh-CN/ja/verified/enter)**才能在交易时不受限制。
- 在开始交易前，请确保您的现货钱包中有足够的资金。

**2. 支持的资产**

- Bitget 提供多种币对（如 BTC/USDT、ETH/BTC）。
- 在“**现货交易”**部分查看其支持币对的完整列表。

**3. 交易机制**

- 订单根据**订单簿**实时匹配。
- 使用**市价单**，以最优可用价格即时执行交易。
- 使用**限价单**指定买入/卖出价格，以便在市场达到该价位时执行。

**4. 交易限制**

- 某些资产或币对可能因监管要求而在某些地区受到限制。请确保在交易前核实您所需的币对是否可用。

**如何在 Bitget App 上查看现货交易手续费？**

1. 点击屏幕左上角的**菜单图标**。

2. 选择“快捷入口”部分底部的“**更多服务”**。

3. 选择“**其他”**选项卡。

4. 点击“费率表”查看您当前的收费等级，该等级由您的交易量和 BGB 持有量决定。

**常见问题1. 现货交易的默认费率是多少？**

挂单和吃单的默认费率均为**0.1%**。

**2. 如何降低我在 Bitget 上的手续费？**
持有 BGB 代币或升级到更高级别的 VIP 可享受折扣。

**3. 现货交易的最小交易量是多少？**
不同交易币对的最小交易量各不相同（例如，BTC/USDT 的最小交易量为 0.0001 BTC）

**4. 现货交易是否有任何限制？**
某些资产可能会受到**地区**法规的**限制**。

**5. 如果我的订单只成交了一部分，该怎么办？**
在执行或取消之前，未成交部分会一直是挂单状态。
