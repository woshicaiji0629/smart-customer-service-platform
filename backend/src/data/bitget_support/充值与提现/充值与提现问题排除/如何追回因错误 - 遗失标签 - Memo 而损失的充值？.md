# 如何追回因错误/遗失标签/Memo 而损失的充值？

*【预计阅读时间：3分钟】*

本文介绍了如何追回在 Bitget 上因标签/Memo 错误或遗失而损失的加密货币充值款项。标签和 Memo 对某些加密货币（如 XRP、EOS、XLM）至关重要，用于确保您的充值入账正确的账户。请按照以下步骤完成追回过程。

**什么是标签或 Memo？标签**或**Memo**是某些加密货币所需的唯一标识符，用于区分您和他人的充值。这些标识符可确保您的资金存入正确的账户。 如果标签或 Memo 有误，您的充值可能无法到账，需要人工操作追回。

**需要标签或 Memo 的加密货币示例：**

- **XRP**：需要接收方标签。
- **EOS**：需要 Memo。
- **XLM**：需要 Memo。

**如何追回充值资金？第1步：访问工单提交表**

1. 点击**[提交请求](https://www.bitget.com/zh-CN/asset/rechargeRefundTagForm)**打开工单提交表单。

**第2步：填写该表格**

1. **查询主题：**选择**“充值”**。

2. **咨询类型：**选择**“链上充值”**。

3. **具体咨询问题：**选择**“我想咨询有关 XRP/EOS/XLM 充值的问题。”**

4. **UID：**输入您的**Bitget 用户 ID**。

5. **电子邮件地址：**请输入有效的邮箱地址

6. **主题：**简要描述您碰到的问题（例如，“XRP 充值缺少标签/Memo”）。

7. **描述：**详尽描述您的问题，包括交易的关键细节。

8. **附件：**如果可能，请上传与您的交易相关截图，以便更快处理。

**第3步：提交请求**

1. 仔细检查所有细节，确保准确性和完整性。

2. 点击“**提交”**完成申请。

**第4步：等待进展**

- **追回过程：**追回款项一般需要**10个工作日**，但复杂情形可能需要更长的时间。

**重要提示**

- **追回费用：**根据问题的复杂程度，追回充值资金可能需要收取手续费。
- **不保证追回：**虽然 Bitget 会尽一切努力追回您的资金，但成功与否取决于交易细节和区块链因素。
- **预防提示：**交易前仔细核对标签/Memo 和充值地址，避免今后出现类似问题。

**常见问题1. 如果我忘记添加标签/Memo 会有什么后果？**

如果缺少所需的标签/Memo，您的充值将无法自动到账。您必须提交手动追回申请。

**2. Bitget 能否追回所有标签/Memo 不正确的充值？**

追回结果因情况而异。尽管大多数充值都有望追回，但实际结果仍取决于交易细节和区块链状态。

**3. 追回过程需要多长时间？**

通常需要10个工作日。复杂的情况可能需要更多时间。

**4. 如果我在追回申请表中输入的信息有误，会发生什么情况？**

不正确或不完整的详细信息可能会延误追回过程。提交表格前，请务必仔细核对信息。
