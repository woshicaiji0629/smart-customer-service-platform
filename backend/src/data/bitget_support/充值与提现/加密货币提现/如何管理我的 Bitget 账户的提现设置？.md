# 如何管理我的 Bitget 账户的提现设置？

*[预计阅读时间：3 分钟］*

本指南全面介绍了Bitget**提现设置**中的主要功能，旨在帮助您提高账户安全性并定制您的提现偏好。无论您是想将取款限制在可信地址，简化小额交易，还是防止未经授权的访问，本指南都将指导您了解每项功能以及如何有效管理它们。

## 提现设置的主要特点

**1. 白名单**

启用**白名单**功能，限制向地址簿中保存的钱包地址取款。该功能可确保资金只发送到受信任的地址，从而增加了一层额外的安全性。

**2. 小额免密提现**

激活**小额免密提现**，允许小额取款（如低于特定 USDT 限额）而无需输入密码。这非常适合频繁、低价值的交易，需要预先配置可信地址。

**3. 撤销提现撤销提现**功能允许您在提交提现后一分钟内取消提现。这有助于纠正错误或阻止未经授权的交易。

**4. 提现跨端确认**

启用**提现跨端确认**功能，以验证在网站上输入的取款地址是否与您的预定地址一致。在应用程序上确认提现，避免地址被劫持。

**如何访问提现设置？App 端：**

1. 登录账户，点击左上角的**菜单按钮**。

2. 点击**个人资料图标**访问**用户中心**。

3. 从菜单中选择**安全中心**。

4. 向下滚动到**提现设置**部分，启用或调整功能。

**网站端：**

1. 点击仪表板右上角的**个人资料图标**。

2. 从下拉菜单中选择**“安全中心”**。

3. 在安全中心中找到**提现设置**部分以配置您的偏好设置。

![如何管理我的 Bitget 账户的提现设置？](https://img.bgstatic.com/multiLang/web/fc075c19a03e8c548c433a8756f42890.png)

**管理退出设置的技巧**

- **保存可信钱包：**使用白名单功能，限制向预先批准的地址取款。
- **有疑问时取消：**启用取消提现功能，以阻止未经授权或不正确的请求。
- **随处验证：**使用跨设备验证来提高安全性，防止地址劫持。
- **从小金额开始：**用**小**金额测试无密码提现，以确保其按预期运行。

**常见问题1. 什么是白名单，为什么要启用白名单？**

白名单功能可将取款限制在地址簿中保存的预先批准的钱包地址，确保资金只被发送到可信的钱包。

**2. 无密码提现如何操作？**

该功能允许在不输入密码的情况下处理预设金额以下的提现。它仅适用于您事先配置的受信任地址。

**3. 提交后可以取消提现吗？**

可以，启用**取消提现**功能后，您可以在提交提现后一分钟内取消提现。

**4. 跨设备提现验证如何提高安全性？**

它确保在网站上输入的取款地址在应用程序上得到验证，防止地址被篡改或劫持。

**5. 我可以同时使用所有提现功能吗？**

是的，启用白名单、无密码提现和跨设备验证等多种功能可确保您提现的最大安全性。
