# 如何查看我的提现状态？

*[预计阅读时间：3 分钟］*

本文将介绍如何在 Bitget 上跟踪加密货币的提现状态。请按照以下步骤快速核实交易进度。

## 如何查看提现状态？

**第 1 步： 进入资产总览部分网页端：**

1. 登录您的[Bitget账户](https://www.bitget.com/zh-CN)，点击右上角的**“钱包”**图标。

2. 选择**资产总览**。

![如何查看我的提现状态？](https://img.bgstatic.com/multiLang/web/d3c7a51dd48d868c53b58a49b65fd5a4.png)

3. 在左侧菜单中选择**钱包充提记录**。

4. 在下拉框中，选择交易类型为**提现**，以查看取款历史记录。

![如何查看我的提现状态？](https://img.bgstatic.com/multiLang/web/7ea6c48241cd979a97843dd1adb586f0.png)

**App 端：**

1. 点击屏幕下方的**“资产”**。

2. 点击**历史记录**的图标。

3. 从下拉菜单中选择**提币**。

![如何查看我的提现状态？](https://img.bgstatic.com/multiLang/web/c07b118b987a5e782ef0189b6512940a.jpeg)

**第 2 步：查看提现状态**

1. 在历史记录列表中找到您的提现。

2. 并提供以下详细信息。

- **日期和时间：**提现申请的时间戳。
- **资产：**您提取的加密货币。
- **数量：**提取的加密货币数量。
- **地址：**发送加密货币的钱包地址。
- **TxID：**交易 ID，用于在区块链上跟踪您的提现。
- **状态：**提现的当前阶段。

**步骤 3：了解提现状态**

- **待处理：**提现正在初步处理中。
- **处理：**正在进行验证或区块链确认。
- **成功：**取款已成功完成并记录在区块链上。
- **失败：**由于钱包地址不正确或资金不足等问题，提现申请未处理。
- **取消：**如果用户在一分钟内成功取消提现，则会显示此状态。一旦取消，资金将保留在用户的 Bitget 账户中。

**常见问题1. 为什么我的提现时间比平时长？**

区块链网络拥堵、系统维护或额外的安全验证都可能导致延迟。您可以使用 TxID 查看区块链上的状态；如果持续延迟，请[提交工单](https://www.bitget.com/zh-CN/ticket/ticket-form?ticketId=2061678148075433984)。

**2. 我可以在区块链上跟踪交易吗？**

是的，您可以使用取款历史记录中提供的 TxID 在区块链浏览器上验证交易状态。

**3. 如果提现状态显示失败，我该怎么办？**

如果取款状态显示失败，请仔细检查取款详情并重试。

**4. 我可以取消提现申请吗？**

不能，提现申请一旦提交并得到处理，就不能取消。在确认之前，请务必仔细查看交易详情。
